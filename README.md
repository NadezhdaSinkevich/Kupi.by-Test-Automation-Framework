# KUPI.BY-Test-Automation-Framework

## О проекте
Данный проект представляет собой автоматизированный тестовый фреймворк для проверки функциональности сайта [kupi.by](https://kupi.by).

## Функционал
Фреймворк обеспечивает комплексное тестирование ключевых модулей системы:
*   **UI Testing:** Автоматизация сценариев взаимодействия пользователя с интерфейсом (форма логина, поиск товаров).
*   **API Testing:** Тестирование программного интерфейса для проверки корректности работы бэкенда (авторизация, поисковые запросы).

## Стек технологий
В проекте использованы следующие инструменты:

*   **Язык программирования:** Java
*   **Тестовый фреймворк:** JUnit 5
*   **Тестирование API:** REST Assured
*   **Логирование:** Apache Log4j
*   **Автоматизация браузера:** Selenium WebDriver
*   **Генерация тестовых данных:** DataFaker
*   **Сборка проекта:** Apache Maven
*   **Контроль версий:** Git
*   **Работа с запросами:** Postman
